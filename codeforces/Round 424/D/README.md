### 题意

有 $n$ 人和 $k$ 把钥匙在一条直线上。每个人想到达一个位于这条直线上的办公室。每个人需要到达钥匙所在的位置拿到钥匙后再去到达办公室。

一把钥匙只能给一个人，无法再给其他人。

你需要确定 $n$ 个人拿到钥匙后再到达办公室的最短时间。我们认为一个人每秒可以跑一个距离。如果2人同时到达了一把钥匙所在的地方，只有一人可以获得钥匙。一个人可以通过一个有钥匙的点并且不取走钥匙。



### Input

第一行包括三个数 $n, k, p (1 \leq n \leq 1000, n \leq k \leq 2000, 1 \leq p \leq 10^9)$ ——人数， 钥匙数， 办公室位置。

第二行包含 $n$ 个不同的整数 $a_1, a_2, ..., a_n (1 \leq a_i \leq 10^9)$ ——人最初位于的位置。这些位置是以任意顺序给出的。

第三行包含 $k$ 个不同的整数 $b_1, b_2, ..., b_k (1 \leq b_j \leq 10^9)$ ——钥匙最初位于的位置。这些位置是以任意顺序给出的。

请注意，一个点不能多于一个人或一把钥匙。一个人和一把钥匙可以位于同一个点上。



### Output

输出 $n$ 个人拿到钥匙后到达办公室使用的最小的时间（秒）



### 题解

读题可知显然钥匙要尽量取中间的，那么可以认为取的钥匙必然是连续的，所以只需枚举作为起点的钥匙，算出最小值即可。
